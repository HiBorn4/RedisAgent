# Screenshots

Place your portfolio screenshots in this directory.

## Required files

| Filename | Dimensions | Description |
|---|---|---|
| `banner.png` | 1280 × 400 px | Hero banner — full app in dark browser, possibly with a blurred Redis logo overlay |
| `chat_ui.png` | 1280 × 800 px | The chat interface with a few messages (user + assistant bubbles visible) |
| `agent_reasoning.png` | 1280 × 800 px | A complex query response showing the agent's key traversal reasoning |
| `health_dashboard.png` | 400 × 600 px | The sidebar showing green "Backend Online" pill, ✓ ADK, ✓ MCP |
| `key_explorer.png` | 1280 × 600 px | Agent response listing Redis keys |

## Optional

- `demo.gif` — Animated GIF (< 5 MB) from your screen recording. Embed directly in README.
- `demo.mp4` — Full quality screen recording (link from README or host on YouTube/Loom).

## Capture tips

- Use **1280px viewport width** for consistency across all screenshots.
- Set browser zoom to **100%** before capturing.
- On macOS: `Cmd + Shift + 4` → drag region → optionally open in Preview to crop/annotate.
- For the banner, consider a simple overlay: background = `#0a0a1a`, title text in gradient red-white, Redis logo bottom-right.
